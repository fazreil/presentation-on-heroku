# Multi-Modules Project

|||
|--|--|
|:white_medium_square:	Manage a group of submodules||
|:white_medium_square:	Located at project's root directory||
|:white_medium_square:	Submodules are regular Maven projects||
|:white_medium_square:	Submodules can be built separately or with aggregator||
|<br/>||
|<br/>||

<Flowchart src="

---

# Multi-Modules Project --> Parent

|||
|--|--|
|:small_blue_diamond:	a POM which its content can be inherited||
|:small_blue_diamond:	1 project/child POM can only have 1 Parent||
|:small_blue_diamond:	child declare parentage in its POM||
|:small_blue_diamond:	Parent is unaware of child that's inheriting||
|:small_blue_diamond:	useful when parent POM consist of reusable content for another project||

---

# Multi-Modules Project --> Aggregator

|||
|--|--|
|:small_orange_diamond: can sequence build process of many projects||
|:small_orange_diamond:	specifies all the projects that can be build-managed together||
|:small_orange_diamond:	child POM unaware of aggregator POM||
|:small_orange_diamond:	Aggregator POM can have more than 1 child POM||
|:small_orange_diamond:	no inheritance of any content from aggregator POM||
|:small_orange_diamond:	useful for modular build & site those projects that should be managed together||
